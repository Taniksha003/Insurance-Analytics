# Insurance Analytics Project

## 📊 Project Overview
This project analyzes insurance data to uncover key trends and insights using Power BI. The focus is on understanding customer demographics, identifying factors influencing claims and premiums, and visualizing the results through interactive dashboards.

---

## 🗂️ Project Structure
- **Insurance.pbix**: Power BI report file containing the analysis and visualizations.
- **InsuranceData.csv**: Dataset used for the analysis.
- 

---

## 🔍 Objectives
1. Analyze customer demographics and distribution.
2. Identify key factors influencing claims and premiums.
3. Provide actionable insights through visualizations.

---

## 🛠️ Tools & Technologies
- **Power BI**: For data visualization and reporting.
- **CSV**: Source format for the data.

---

## 📈 Key Insights
- **Customer Demographics**:
  - Age and gender distribution of policyholders.
  - Regional trends in insurance claims.

- **Claims Analysis**:
  - Key drivers of claim frequency.
  - Correlation between premium amounts and claims.

- **Financial Metrics**:
  - Total premiums collected.
  - Claims vs. revenue trends.

---

## 📊 Dashboards
Here are some of the key visualizations from the project:

1. **Demographics Dashboard**: Age, gender, and region-wise analysis.
2. **Claims Analysis Dashboard**: Factors influencing claims and premium trends.
3. **Financial Performance Dashboard**: Visualizing profitability and revenue streams.
---

## 📂 Dataset
- **Format**: CSV file containing anonymized insurance data.
- **Columns**:
  - `CustomerID`: Unique identifier for customers.
  - `Age`: Age of the customer.
  - `Gender`: Gender of the customer.
  - `Region`: Location of the customer.
  - `Premium`: Insurance premium amount.
  - `Claims`: Number of claims made.

---

## 🚀 Getting Started
1. **Download the Files**:
   - Download `Insurance.pbix` and `InsuranceData.csv`.

2. **Open in Power BI**:
   - Import the `Insurance.pbix` file into Power BI Desktop.
   - Ensure `InsuranceData.csv` is in the same directory or update the data source path.

3. **Explore the Dashboards**:
   - Use Power BI's interactive features to explore trends and insights.
